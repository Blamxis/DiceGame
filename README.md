# Dice Game Evalutaion JavaScript . 

# Rules:
# Pour jouer à ce petit jeu de dé, vous devez être deux joueur.
# Les règles sont simple, vous avez 3 boutouns sur le tableau de jeu (Roll Dice, Hold ainsi que New Game).
# Au début de la partie, c'est le joueur 1 qui commence, il doit cliquer sur Roll dice afin de faire apparaitre une face du dé qui lui donnera un score.
# Ce score sera enregistrer dans le score actuel ou current. Le joueur peut jouer autant de fois qu'il le souhaite tant qu'il ne fais pas de 1.
# Si le joueur fais un 1 son score actuel retombe à 0 et son tour est terminé, c'est maintenant au joueur 2 de cliquer sur roll Dice.
# Si vous pensez avoir assez de points, il suffit de cliquer sur Hold pour sauvegarder votre score et ca passe ensuite au prochain joueur.
# Appliquer ces règles jusqu'a ce qu'un joueur atteigne les 100 points dans ce cas le joueur gagnant aura le titre de Winner tandis que l'autre joueur sera Loser.
# En cliquant sur New Game, le deé se reinitialise sur la face 1 et les scores reviennent à 0 ainsi qu eles titres qui disparaissent.
# Petit plus, vous ne pouvez pas cliquer sur Hold tant que vous n'avez pas cliquer sur Roll.
